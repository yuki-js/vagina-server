## How to Use

We recommend attempting exercises **before** looking at answers. Use these files:

1. **Hints first** - If you're stuck, read the hint section
2. **Partial answers** - For verification after attempting
3. **Complete answers** - Only as last resort

## Don't Spoil Your Learning

Every time you look at an answer without trying, you reduce your learning. The struggle is part of the process.

> "I don't know" is the starting point of learning.
> "I looked it up" is the starting point of forgetting.
